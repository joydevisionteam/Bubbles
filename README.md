# Bubbles
A minimal shooting game written in JavaScript and [p5.js](http://p5js.org).

Play [here](http://dfilaretti.github.io/software/bubbles)! 

If you want to learn JavaScript in a creative setting, I reccomend this [video tutorial series](https://www.youtube.com/playlist?list=PLRqwX-V7Uu6Zy51Q-x9tMWIv9cueOFTFA).
